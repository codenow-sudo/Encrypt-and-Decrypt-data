Steps to Follow:

1) Place the '.txt' file which need to be encrypted in this folder
2)Rename it to '1.txt'
3)Run 'key_gen.exe' to generate key. It will create 'key.txt' file.
4)Run 'encrypt.exe' and wait untill it is done ( It takes less than 1 min) and a new file with name '1_encrypted.txt' is created.
5)Encrypt Done.....................................

TO DECRYPT IT :

6)Run 'decrypt.exe' and wait untill it is done ( It takes less than 1 min) and a new file with name '1_decrypted.txt' is created.
5)Decrypt Done.....................................
